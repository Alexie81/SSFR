# Prior-art checklist

Search each query across arXiv, DBLP, ACM Digital Library, IEEE Xplore, PVLDB,
OpenReview, Google Patents, Lens, Semantic Scholar, and Crossref:

- Fourier vector database routing
- FFT shard selection
- spectral shard routing
- frequency-domain ANN routing
- compressed centroid scoring
- Fourier centroid search
- certified spectral routing
- adaptive Fourier nearest neighbor
- spectral partition probing
- FFT distributed vector search

For every relevant result record title, authors, year, venue, stable identifier,
claimed contribution, relation to SSFR, and whether it invalidates or narrows the
working novelty statement. Also review adjacent work on learned routing,
quantization, IVF/HNSW partition probing, spectral seriation, fast transforms for
MIPS, certified ANN, and branch-and-bound shard pruning.

Do not infer novelty from the absence of the name “SSFR”.
